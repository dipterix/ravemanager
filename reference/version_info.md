# Print out 'RAVE' version information

Print out 'RAVE' version information

## Usage

``` r
version_info(vanilla = FALSE, auto_restart = !get_os() %in% "windows", ...)
```

## Arguments

- vanilla:

  whether to use vanilla packages in this function

- auto_restart:

  whether to automatically install `ravemanager` if an update is
  available and apply restart; default is true on non-Windows machine

- ...:

  reserved for future use
